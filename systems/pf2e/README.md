# cruncher-pf2e

Pathfinder 2e Remaster SRD system pack for [cruncher](https://github.com/erenes/lorekit).

Contains game mechanics and rules text derived from the Pathfinder
Second Edition Remaster SRD, licensed under the
[ORC License](https://paizo.com/orclicense).

Only openly licensed game mechanics are included. No proprietary
content, artwork, storylines, or Reserved Material is distributed
in this package.

## Usage

```python
import cruncher
import cruncher_pf2e

pack = cruncher.load_system_pack(cruncher_pf2e.pack_path())
```
