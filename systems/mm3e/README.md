# cruncher-mm3e

d20 Hero SRD (3e) system pack for [cruncher](https://github.com/erenes/lorekit).

Contains game mechanics and statistics derived from the
[d20 Hero SRD](https://www.d20herosrd.com/), licensed under the
Open Game License v1.0a.

Only openly licensed game mechanics are included. No proprietary
content, artwork, storylines, or Product Identity is distributed
in this package.

## Usage

```python
import cruncher
import cruncher_mm3e

pack = cruncher.load_system_pack(cruncher_mm3e.pack_path())
```
